什么是 Frontend-Slides？

Frontend-Slides 是由张咋啦原创的 AI 演示文稿生成工具，在小红书获得 2.4 万赞的爆款工具。

零代码

快速生成

Install
03
三步完成安装

安装步骤

在 Claude Code 中输入 /frontend-slides

🔗 GitHub 仓库：https://github.com/zarazhangrui/frontend-slides

  • 第一步：确保电脑已安装 Claude Code 或支持的 IDE（如 VS Code）
  • 第二步：在终端中执行：git clone https://github.com/zarazhangrui/frontend-slides.git ~/.claude
  • 第三步：在 Claude Code 中输入 /frontend-slides，出现功能提示即安装成功

Step 3
07
在线编辑和调整

内容编辑阶段

编辑模式（左上角编辑按钮）
细节调整
  • 在线编辑：悬停在左上角或按 E 键进入编辑模式，点击任何文字直接修改
  • 自动保存：所有修改自动保存到浏览器本地存储，随时恢复
  • 对话调整：需要大改动时，直接告诉 AI，如"换一个更鲜艳的配色"
  • 导出分享：完成后可部署为在线链接或导出为 PDF
